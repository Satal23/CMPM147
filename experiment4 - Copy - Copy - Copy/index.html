Experiment 2 - Generative Landscape

Description

I was trying to capture the image specifically the general shapes in the image. For example the shapes of the trees and the shapes of the mountain.

Technical

I used perlin noise to generate a randomized continious line representing the terrain height. The function calculates y coordinate points along the terrain line using perlin noise and puts them in an array for later use. There are frequency and amplitude parameters used to control the scale and roughness of the terrain. I also used random generation for trees and clouds so that they are placed randomy. The rendering is done in the drawScene() function. In this function I drew some of the elements such as the gradient background, shape of the terrain, clouds and trees. The trees are made using triangles and rectangles.

Reflection

All in all it was a fun project to work on. I worked on it alone so I was responsible for all of the code though i did have some help from examples and chatgpt on perlin noise generation.
